
# Scala Text Analyzer

Author: **Sai Geethika**

## Requires Java 17+ to run properly

This project requires **Java 17** (LTS) or newer. Please ensure you have Java 17 installed before running the application.

### How to Install Java 17

1. Download Java 17 from [Adoptium](https://adoptium.net/).
2. Choose:
	- **Version:** 17
	- **Installer:** MSI
	- **Architecture:** x64
3. Run the installer and follow the instructions.
4. Verify installation by running:
	```powershell
	java -version
	```
   You should see output similar to:
	```
	openjdk version "17.0.x" ...
	```

If Java 17 is not installed, the application may fail to run or download content may be empty.

## Overview
This project demonstrates a clean and human-readable Scala application
that performs text analysis on a remote file.
